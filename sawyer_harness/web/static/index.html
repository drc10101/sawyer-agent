Settings

Main Model

The LLM that powers the main agent. Changes take effect on the next message.

Agent Mode

Direct answers you itself, uses its own tools. Fast, cheap, good for most conversations. Orchestrator decomposes goals into subtasks, delegates to sub-agents, evaluates results. Slower but more capable for complex work. Auto starts direct, escalates to orchestration when needed.

Response Style

Controls how verbose the agent's responses are. Concise gives short answers with no tool output. Normal is balanced. Thorough explains reasoning in detail.

Max Tool Rounds

Maximum number of tool-call loops before the agent stops. Lower values prevent runaway tool usage for simple questions.

20

Show Tool Output

When enabled, tool results are streamed into the chat as they arrive. Disable for cleaner output.

Agreeability

Agreeable tells you what you want to hear. Balanced is honest but tactful. Honest always tells the truth, even when it stings.

Reasoning Depth

Low answers fast with minimal explanation. Medium shows key steps. Medium-High considers edge cases. High shows full reasoning chain and verifies each step.

Version Control

Mark a version as "last known good" when things are working. Revert to it if something breaks.

Session Scores

Rate sessions to track agent quality over time. Higher honesty scores mean the agreeability setting is working.

Server

Restart the server to apply code changes or config updates. Your data is preserved.

Vision Bridge -- Google AI Studio API Key
This key powers the vision bridge -- the agent's ability to see and verify web pages, screenshots, and UI output. Without it, the agent is blind: it cannot check whether things look correct.

How to get it:
  1. Go to ai.google.dev
  2. Sign in with a Google account (free)
  3. Create a project and generate an API key
  4. Paste the key below

The free tier includes generous usage. The key is stored locally and never sent anywhere except Google's API for vision checks.
